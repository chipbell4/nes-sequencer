(static) Arpeggio(frequencies, cycles, startVolume, endVolume) → {Array}

Takes a chord of frequencies and converts them into an arpeggiated chords that arpegiates at 60Hz (60 notes per second, the NES limit)
Parameters:
Name Type Description
frequencies Array An array of floating-point frequencies
cycles Number The number of cycles to run
startVolume Number The starting volume (0 - 1)
endVolume Number The ending volume (0 - 1), defaults to the starting volume
