(static) play(melodies, loop)

Schedules a melody to be played. melodies, in this case is an object, where the keys are the oscillator type ('PWM1', 'PWM2', 'NOISE', 'TRIANGLE'), and the associated values is an array of objects, each looking like this: { frequency: 440, volume: 0.2, cycles: 10 }, where frequency is in Hz, volume ranges between 0 and 1, and cycles is the number of cpu cycles to use
Parameters:
Name Type Description
melodies Array The melody to play
loop Boolean Whether or not to loop the song. Defaults to false
